Sample Article for Extension Testing

This is a test page for the Chrome extension. It contains selectable text that will trigger the floating action buttons (FABs) when selected.

Artificial intelligence is transforming how we interact with technology. Machine learning models can now understand context, generate creative content, and assist with complex problem-solving tasks. This technological revolution is reshaping industries from healthcare to education, making previously impossible tasks routine.

The extension should detect when text is selected and show two emoji buttons: ✨ for summarization and ✒️ for drafting responses. These buttons use the latest AI models to process the selected text.
