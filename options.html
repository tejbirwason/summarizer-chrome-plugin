Summarizer Extension Options

Cache Management

Summaries are cached to save API calls and improve performance. Cached summaries are automatically removed after 30 days.

Cloud Worker

Summaries, posters and history live in a Cloudflare Worker. Paste the endpoint and app token once — no Python, no native host.

About

This extension provides AI-powered summarization using OpenAI's o3 model for web pages and YouTube videos.

Storage Keys:
• YouTube: summary:yt:{videoId}
• Web pages: summary:page:{url}
